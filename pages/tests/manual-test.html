Manual Testing Steps

Test 1: Example Cards Click

  1. Open browser console (F12)
  2. Go to http://localhost:5173
  3. Close onboarding modal if it appears
  4. Run in console: document.querySelector('#http500Rate').value (note the value)
  5. Click on "API Meltdown" card
  6. Run in console again: document.querySelector('#http500Rate').value
  7. Value should have changed to 30

Test 2: Check if applyFaultConfiguration exists

  1. In browser console at http://localhost:5173
  2. Run: typeof applyFaultConfiguration
  3. Should return "function"
  4. If undefined, the function isn't being loaded

Test 3: Check example data

  1. In browser console:
  2. Run: document.querySelectorAll('#examplesGrid .ex').forEach((el, i) => { console.log(i, el.getAttribute('data-example-index'), el.querySelector('h3').textContent); })
  3. Should show 6 examples with indices 0-5

Test 4: Manual fault application

  1. In browser console:
  2. Run: if (typeof applyFaultConfiguration === 'function') { applyFaultConfiguration({ http500Rate: 0.5, malformedRate: 0.8 }); console.log('Applied!'); } else { console.log('Function not found!'); }
